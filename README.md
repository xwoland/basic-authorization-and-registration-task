1. Настроить vhosts.
2. Создать новую базу данных и заменить значение константы DB_DATABASE в .env файле на имя новой бд.
3. Применить команду php artisan migrate
4. В корне проекта лежит json-файл Basic auth and registration task.postman_collection.json с коллекцией для Postman. Его нужно импортировать в вашем Postman-клиенте, исправить URL на ваш и отправить запрос.

Общее время работы - 3 часа 10 минут.
